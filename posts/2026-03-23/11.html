11. Sun, 22 Ma｜獨家參觀 Amazon Trainium 實驗室，贏得了 Anthropic、OpenAI 甚至 Apple 的青睞

全文翻譯（繁體中文・香港）

亞馬遜執行長 Andy Jassy 宣布 AWS 與 OpenAI 達成突破性的 500 億美元投資協議後不久，亞馬遜邀請我私人參觀了該協議核心的晶片開發實驗室，費用（大部分*）是亞馬遜自費。

產業專家正關注亞馬遜在該工廠生產的 Trainium 晶片，看其對低成本人工智慧推理的影響，並有可能削弱英偉達近乎壟斷的地位。

我當天的導遊是實驗室主任 Kristopher King（右下圖）和工程總監 Mark Carroll（左下圖），以及安排這次訪問的團隊公關人員 Doron Aronson（在故事後面與您的真實合影）。

自 AI 實驗室成立之初起，AWS 就一直是 Anthropic 的主要雲端平台——這種關係足以讓 Anthropic 得以生存，後來 Anthropic 也將微軟加入為雲端合作夥伴，而且亞馬遜與 OpenAI 的合作夥伴關係也不斷發展。

OpenAI 交易使 AWS 成為模型製造商新的 AI 代理構建器 Frontier 的獨家提供商，如果代理變得像矽谷認為的那樣大，那麼 Frontier 可能會成為 OpenAI 業務的重要組成部分。我們將看看這種排他性是否與所宣布的完全一致。英國《金融時報》本週報導，微軟可能認為 OpenAI 與亞馬遜的交易違反了它自己與 OpenAI 的協議，即微軟獲得了 OpenAI 的所有模式和技術的存取權。

是什麼讓 AWS 對 OpenAI 如此有吸引力？作為交易的一部分，這家雲端巨頭已同意向 OpenAI 提供 2 吉瓦的 Trainium 運算能力。這是一個巨大的承諾，因為 Anthropic 和亞馬遜自己的 Bedrock 服務消耗 Trainium 晶片的速度已經超過了亞馬遜生產它們的速度。

該公司表示，所有三代都部署了 140 萬個 Trainium 晶片，Anthropic 的 Claude 在部署的超過 100 萬個 Trainium2 晶片上運行。

值得注意的是，雖然 Trainium 最初是針對更快、更便宜的模型訓練（幾年前更重要），但現在它也經過調整並用於推理。推理——實際運行人工智慧模型以產生回應的過程——是目前業界最大的效能瓶頸。

舉個例子：Trainium2 處理亞馬遜 Bedrock 服務上的大部分推理流量，該服務支援亞馬遜許多企業客戶建立人工智慧應用程序，並允許應用程式使用多種模型。

金說：「我們的客戶群正在以我們能夠提供的能力盡可能快的速度擴大。」「有一天，Bedrock 可能會像 EC2 一樣大，」他補充道，他指的是 AWS 龐大的運算雲端服務。

亞馬遜表示，除了為 Nvidia 積壓的、難以獲得的 GPU 提供替代品之外，亞馬遜還表示，在其新的專業 Trn3 UltraServers 上運行的新晶片與使用傳統雲端伺服器相比，在同等性能​​下運行成本可降低 50%。

除了 12 月發布的 Trainium3 之外，該 AWS 團隊還建立了新的 Neuron 交換機，Carroll 表示該組合具有變革性。

「這給我們帶來了巨大的好處，」卡羅爾說。這些交換器允許每個 Trainium3 晶片與網狀配置中的每個其他晶片通信，從而減少延遲。他說：“這就是為什麼 Trainium3 打破了各種記錄”，特別是“單位價格”。

當每天涉及數萬億代幣時，這種改進就會累積起來。

事實上，亞馬遜的晶片團隊在2024年就受到了蘋果的稱讚。在這家神秘公司罕見的公開時刻，蘋果的人工智慧總監公開描述了它如何使用該團隊的另一款晶片——Graviton，一種低功耗、基於 ARM 的伺服器 CPU，也是該團隊設計的首款突破性晶片。蘋果也稱讚了 Inferentia（一款專門為推理而設計的晶片），並對當時的新產品 Trainium 表示認可。

這些晶片代表了亞馬遜的經典策略：了解人們想買什麼，然後建立一個在價格上競爭的內部替代品。
