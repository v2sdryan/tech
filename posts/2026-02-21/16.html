16. ⭐ 2026-02-21 GitHub AI 焦點：pentagi — 全自主 AI 滲透測試系統登上趨勢榜

全文翻譯（繁中）

由 vxcontrol 團隊開發的 pentagi 項目近日登上 GitHub 趨勢榜，引發了網絡安全社群的廣泛關注。pentagi 是一個全自主的 AI 驅動滲透測試系統，能夠在無需人類指導的情況下自動規劃、執行和報告完整的安全測試流程。該系統結合了大型語言模型的推理能力和傳統滲透測試工具的技術能力，代表了 AI Agent 在網絡安全領域的前沿應用。項目在 GitHub 上線後短短一週內已累積超過 5,000 顆星，Fork 數超過 800。

pentagi 的技術架構採用了多 Agent 協作的設計模式。系統由五個專門化的 AI Agent 組成：「偵察 Agent」負責收集目標系統的公開信息和攻擊面；「漏洞分析 Agent」基於偵察結果識別潛在的安全弱點；「漏洞利用 Agent」嘗試利用已識別的弱點獲取系統訪問權限；「提權 Agent」在獲得初始訪問後嘗試擴大權限範圍；「報告 Agent」則將整個測試過程和發現的安全問題整理成專業的安全評估報告。這五個 Agent 透過共享的任務管理系統進行協調，形成了一條完整的自動化攻擊鏈。

在實際測試中，pentagi 展現出令人印象深刻的能力。根據項目文檔中披露的基準測試結果，pentagi 在 HackTheBox 的 20 個「Easy」難度靶機中成功攻克了 18 個（90% 成功率），在「Medium」難度靶機中成功攻克了 12 個（60% 成功率），在「Hard」難度靶機中成功攻克了 4 個（20% 成功率）。雖然在高難度場景中的表現仍不及資深人類滲透測試員，但 pentagi 的優勢在於速度和一致性——它可以 24 小時不間斷地運行，且不會因疲勞或疏忽而遺漏測試步驟。

pentagi 的出現引發了安全社群內部的激烈辯論。支持者認為，AI 自動化滲透測試可以大幅降低安全評估的成本和時間門檻，讓更多中小企業負擔得起定期的安全測試——目前一次專業的人工滲透測試報價通常在 10,000-50,000 美元之間，而使用 pentagi 的成本（主要是 AI API 調用費用）僅為數百美元。反對者則警告，同樣的技術如果被惡意行為者使用，將大幅降低網絡攻擊的技術門檻，使「腳本小子」也能發起以往只有高級黑客才能執行的複雜攻擊。

從法律和倫理角度，pentagi 的開源發佈引發了關於「攻擊性安全工具」邊界的討論。項目的 README 文件中包含了醒目的法律免責聲明，強調 pentagi 僅供授權的安全測試使用，未經授權對他人系統進行滲透測試是違法行為。然而，開源代碼一旦發佈便無法控制其使用方式。多位安全專家建議，AI 滲透測試工具的開發應遵循「負責任披露」原則，並考慮在系統中內建限制機制——例如要求用戶提供授權證明才能對特定 IP 範圍進行測試。對於香港的企業安全團隊而言，pentagi 是一個值得關注的工具，可作為人工滲透測試的補充手段，但在使用時必須確保嚴格遵守香港《電腦罪行條例》的相關規定。
