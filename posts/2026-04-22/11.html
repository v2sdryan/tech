11. 2026-04-21｜Zorin OS 18.1 發布 - Lite 版本再次出現

全文翻譯（繁體中文・香港）

Zorin OS 的最新版本已經發布，對於仍在尋找 Windows 10 替代品的人來說，它是 Linux Mint 的一個有趣的替代品。

Zorin OS 18.1 已發布。這是該公司基於 Ubuntu Noble 的系列的第一個單點版本，在 Zorin OS 18 發布大約六個月後發布，我們在去年 9 月看到了 Zorin OS 18 的測試版本。此版本吸收了 2 月的 Ubuntu 24.04.4 的更新，包括來自 Ubuntu Questing Quokka 版本的核心 6.17。

隨後的 Ubuntu LTS 版本 26.04 將於本週稍後發布，上個月我們研究了 Resolute Raccoon beta 版。不過，Zorin 卻按照自己的步調行事，並制定了在新版本準備就緒時發布新版本的政策。除了從 Ubuntu 繼承的核心作業系統元件之外，18.1 版本還擁有自己的其他更新，例如最新版本的 LibreOffice 26.2 版以及對桌面環境的許多自訂的新版本。

公告稱，該版本擁有更廣泛的硬體支援和更好的效能，這主要得益於上游 Ubuntu LTS 支援堆疊的更新元件。桌面仍然使用相同的 GNOME 版本 - 特別是版本 46.7 - 但 Zorin OS 通過一套 GNOME 擴展對其進行了顯著定制 - 例如，該公司贊助了 Dash-to-Panel 擴展的開發。在此版本中，有 27 個單獨的擴充。除其他改進外，還改進了視窗平鋪，提供更多佈局選項，只需單擊即可將相關視窗群組置於前面。該面板現在提供針對從右到左閱讀的腳本（例如阿拉伯語和希伯來語）的佈局，這些佈局顛倒了面板中組件的順序。此外，在面板中，可以開啟和關閉狀態圖示。

Zorin OS 提供了 Linux 世界中一些對 Windows 應用程式最好的處理能力。這隻禿鷹過去曾多次向朋友提供 Linux 機器，而我們不只一次看到天真的用戶試圖下載 Windows 應用程式。 Zorin OS 桌面會嘗試識別這些下載，如果存在本機 Linux 版本，則建議安裝這些版本。如果沒有本機版本，它將建議合適的替代品或替代品。如果這也不是一個選項，它會為您提供安裝 WINE 10，然後嘗試以這種方式安裝應用程式。在此版本中，它可以處理的 Windows 應用程式清單增加了 40%。

儘管這只是一個小版本，但 Zorin OS 18.0 的一個變化對於某些用戶來說意義重大：Zorin OS Lite 的回歸。 Zorin OS Lite 看起來與旗艦 Zorin OS 發行版非常相似，但 Lite 版本不是使用 GNOME 桌面，而是使用 Xfce。去年版本 18 出現時，僅更新了 GNOME 版本，不包括 Xfce 版本。 Zorin OS Lite 唯一可用的版本是先前的版本，即基於 Ubuntu Jammy 的 Zorin OS 17.3 。在 18.1 中，Xfce 版本重新加入了基於 GNOME 的兄弟版本：免費的 Core 和 Education 版本以及付費的 Zorin OS Pro。

新的 Zorin OS 18.1 Lite 使用 Xfce 4.20。它於 2024 年底發布，是當前版本，特別是比 Zorin 的上游 Ubuntu 24.04“Noble Numbat”中包含的版本更新，後者仍然提供 2022 年的 Xfce 4.18 。我們很高興看到 Zorin OS Lite 的回歸有幾個原因。

首先是我們對 Xfce 日益增長的熱愛。在過去的幾年裡，這位禿鷹在 Reg FOSS 辦公桌上工作，嘗試了跨多個 FOSS 類 Unix 作業系統的許多不同的 FOSS 桌面環境，而 Xfce 是我們最喜歡的。它達到了卓越的功能、豐富的客製化、緩慢、可衡量的發布週期以及相對較小之間的最佳點。這一次，你會得到「好」、「快」（在效能方面）和「便宜」（在輕資源使用的意義上）。

其次，它反駁了我們看到的針對 Xfce 最常見的指控之一——它看起來沉悶、過時或無聊。 Zorin OS 版本表明事實並非如此：Xfce 看起來很棒，而且在這個版本中，它大放異彩。我們認為，Zorin OS 在任何作業系統上都提供了最時尚的 Xfce 配置。它看起來與旗艦 GNOME 版本幾乎相同。 GNOME 的圖形設計非常出色，Zorin 在此基礎上提供了明亮、高對比度的主題和有吸引力的壁紙。

遺憾的是，該公司計劃「淘汰」Zorin OS Lite 的聲明仍然存在。未來的 Zorin OS 19.x 發布週期將僅限於 GNOME。我們對此感到左右為難。就個人而言，這隻禿鷹更喜歡 Xfce，而 Zorin 的此桌面配置是其最好的展示之一。另一方面，專注於一個桌面環境將減少公司的工作量並簡化支援工作量。該公司觀察到：

現在可以在 RAM 低至 2 GB 的電腦和使用時間長達 15 年的電腦上執行 Zorin OS 的非 Lite 版本。

這是公平的。即使是 2008 年的硬體現在也能夠很好地運行 GNOME：一年前，當我們在 Core 2 Duo 上測試 GNOME 48 時，它的運作效果給我們留下了深刻的印象。沒有 32 位元版本，因此它對真正的老年套件沒有幫助，但即使是 20 年前的高端套件也有很好的機會運行得這麼好。

不過，新的 Zorin OS Lite 仍然基於 Ubuntu Noble，這意味著它將與我們去年初強調的舊版 Nvidia GPU 有相同的硬體相容性問​​題。如果您的 Nvidia GPU 硬體需要該公司的任何舊版驅動程式（即 470 或更早版本），它們可能無法與 Zorin OS Lite 18.1 配合使用，或者只能透過通用的 Nouveau 驅動程式運作。

這三個免費版本都非常相似：Lite 有 Xfce，Core 和 Education 有 GNOME，並且都提供類似的 Windows 或類似 GNOME 的佈局選擇，以及非常簡單的 Zorin Appearance 應用程序，以簡化它們之間的切換。它們都包含一系列標準 Linux 應用程序，包括 Brave 瀏覽器、媒體和圖像處理工具，以及一些方便的附加功能，例如用於連結到智慧型手機的 Zorin Connect 工具。

付費專業版以 Flatpak 形式捆綁了大量領先的 FOSS 應用程式、多個附加桌面佈局，更有價值的是，它還包括客戶支援。定價與先前發布的版本相同：無論是美元、歐元還是英鎊，均為 47.99。任何配備 Windows 10 的機器都應該運作得很好，我們預計它在相同硬體上比 Windows 快得多。

Zorin OS 與 Linux Mint 一樣是取代 Windows 的主要競爭者之一。 Zorin OS 進一步嘗試讓 Windows 遷移者有賓至如歸的感覺，雖然它提供的桌面較少，但它擁有更多的自訂工具和明亮的外觀主題。當然，旗艦版也是要花錢的。
