01. 2026-03-23｜微軟：刪除部分副機長將改善Windows 11

全文翻譯（繁體中文・香港）

微軟承認需要提高 Windows 11 的質量，並概述了完成這項工作的計劃。

Windows 和設備部門執行副總裁 Pavan Davuluri 在周五發給 Windows Insider 的帖子中表示，他在用戶中進行了調查，“收到的是那些非常關心 Windows 並希望它變得更好的人們的聲音。”

達烏魯裡表示，微軟已經開始改進作業系統，並提供了一系列更改，首先是「更多任務欄定制，包括垂直和頂部位置」。

這是 Microsoft 30 年前在 Windows 95 中提供的功能。

我們正在減少不必要的副駕駛入口點

他的下一個專案是“透過技巧和專注，將人工智慧整合到最有意義的地方”，讀起來很像承認微軟在 Windows 中的人工智慧方面犯了錯誤。

「你會看到我們更加重視 Copilot 在 Windows 上的整合方式和位置，專注於真正有用且精心設計的體驗，」Davuluri 寫道。 “作為其中的一部分，我們正在減少不必要的 Copilot 入口點，從截圖工具、照片、小部件和記事本等應用程式開始。”

Davuluri 還承諾，文件資源管理器將變得“更快、更可靠”，為日常文件任務提供“更快的啟動體驗、減少閃爍、更流暢的導航和更可靠的性能”。他還承諾“複製和移動大型文件將更快、更可靠”，並且“搜尋、導航和上下文選單的延遲將大大降低”。

另一項計劃呼籲“降低 Windows 的基準內存佔用量，為您運行的應用程序釋放更多容量。”該操作系統將能夠“即使在負載下也能提供更一致的性能，因此應用程序可以全天保持響應。”

這可以說不是對用戶回饋的回應，而是商業上的需要，因為記憶體價格的飆升意味著僅 RAM 就佔據了 PC 價格的三分之一以上，導致購買者的成本更高。

如果個人電腦銷售放緩，微軟的收入也會放緩。因此，讓 Windows 11 適應僅配備 8GB RAM 的機器已成為常態的時代，是企業的當務之急，而不僅僅是軟體巨頭作業系統工程師面臨的技術挑戰。

微軟待辦事項清單上的其他項目解決了人們對 Windows 的老問題，例如連接藍牙設備時性能不穩定、電腦從睡眠中醒來後脾氣暴躁或緩慢，以及減少小部件的侵入性。 Davuluri 也表示，微軟將努力加快 Windows 安裝速度，並將為 Windows Update 提供每月重新啟動一次的選項。

這位高層還承諾透過一個新的、更易於使用的回饋中心來改進 Windows Insider 計劃。同樣，這將取悅用戶並使微軟受益。
