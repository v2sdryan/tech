05. 2026-04-17｜Anthropic 模仿 Claude Design 為行銷團隊起草精美的新解僱通知書

全文翻譯（繁體中文・香港）

Anthropic 以其業界領先的 Claude Code 編寫程式而聞名，但為什麼止步於此呢？該公司週五推出了一項名為 Claude Design 的研究預覽服務，該服務可以創建視覺資產，可能會讓一些人失業。

Claude Design 消息一出，設計公司 Figma 的股價就下跌了 7% 左右。 Claude Design 也代表了人工智慧設計服務 Lovable 的一次嘗試。

「Claude Design 為設計師提供了廣泛探索的空間，也為其他人提供了一種製作視覺作品的方式，」Anthropic 在一篇部落格文章中說道。 “描述您的需求，Claude 會建立第一個版本。從那裡，您可以透過對話、內聯評論、直接編輯或自訂滑桿（由 Claude 製作）進行改進，直到正確為止。”

Anthropic 預計 Claude Design 將有助於設計原型、產品線框圖和模型、設計探索、宣傳材料和演示以及行銷材料。

該服務包括建立您自己的設計系統的選項。這涉及提供 GitHub 儲存庫、本機程式碼檔案、上傳的 Figma 檔案、包含字體、徽標和其他資產的資料夾以及指導底層 LLM 的文字註釋的連結。

此後，項目會繼承此樣式訊息，因此設計不會從頭開始。

與 Claude Design 合作需要的是製作提示的能力，而不是熟練的手工操作和一些藝術教育。 「你不需要成為設計師才能獲得出色的成果，」公司文件宣稱。 “具體說明你正在構建什麼，它是為誰構建的，以及什麼是最重要的。”

對話式設計過程結束後，使用者可以選擇以各種格式（.zip、.pdf、.pptx）下載結果，或將結果匯出為 Canva、HTML 或 Claude Code。

Claude Design 的使用與其他 Claude 服務分開計量和追蹤。 「它有自己的使用追蹤、自己的限額，以及對於訂閱計劃而言，它自己的每週限制，與您現有的聊天或 Claude Code 限制並存（而不是在其中），」Anthropic 解釋道。

AI 企業正在以一次性積分的形式為基於企業使用的 Claude Design 用戶提供體驗，據說涵蓋了約 20 個典型提示。該積分將在額外的 Claude Design 使用量計入組織支出之前消耗，並於 7 月 17 日到期。

就在一個月前，在 Reddit 的一個平面設計論壇上，有人發了一篇帖子，上面寫著：“沒有人工智慧，你就無法拼寫‘下崗’。”

然而，人工智慧將取代創意工作的說法可能過於簡單化。

舊金山灣區的平面設計師 Molly McCoy 在接受採訪時告訴 The Register，她並沒有真正使用人工智慧工具做太多事情。

「我並沒有真正使用人工智慧，」麥考伊說。 「我作為專業平面設計師已經有二十五年了，我將其視為一種工具。但對於我正在做的印刷工作來說，一切都必須在足球場大小的印刷機上進行，而人工智慧實際上更加理論化。

「我一直在用我的大腦來做人工智慧所做的一些事情，當我涉足它時，我就想，『哦，這就像一台不會中獎的老虎機。』”

她說她的客戶對人工智慧工作感到滿意。 “所以我對此印象深刻，但對於我自己的工作流程來說，它沒有任何存在或影響。”
