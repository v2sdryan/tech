20. 2026-03-09｜亞洲關鍵基建受攻擊：涉 Web 漏洞利用與 Mimikatz

全文翻譯（繁體中文・香港）

作為長達數年活動的一部分，位於南亞、東南亞和東亞的高價值組織已成為中國威脅行為者的目標。

該活動針對航空、能源、政府、執法、製藥、技術和電信部門，Palo Alto Networks Unit 42 將其歸因於之前未記錄的威脅活動組織 CL-UNK-1068，其中“CL”指“集群”，“UNK”代表未知動機。

然而，安全供應商以「中等到高置信度」評估活動的主要目標是網路間諜活動。

安全研究員 Tom Fakterman 表示：“我們的分析揭示了一個多方面的工具集，其中包括自定義惡意軟體、修改後的開源實用程式和非本地二進製文件 (LOLBIN)。” “這些為攻擊者提供了一種簡單、有效的方法來在目標環境中保持持久存在。”

這些工具旨在針對 Windows 和 Linux 環境，攻擊者依賴開源實用程式和惡意軟體系列的組合，例如 Godzilla、ANTSWORD、Xnote 和快速反向代理程式 (FRP)，所有這些都已被各種中國駭客組織使用。

雖然 Godzilla 和 ANTSWORD 都充當 Web shell，但 Xnote 是一個 Linux 後門，自 2015 年以來就在野外被檢測到，並由名為 Earth Berberoka（又名 GamblingPuppet）的敵對團體部署，用於針對在線賭博網站的攻擊。

典型的攻擊鏈需要利用 Web 伺服器來傳遞 Web shell 並橫向移動到其他主機，然後嘗試從 Windows Web 伺服器的「c:\inetpub\wwwroot」目錄中竊取與某些副檔名（「web.config」、「.aspx」、「.asmx」、「.asax」和「.dll」）相符的文件，可能是為了竊取憑證。

CL-UNK-1068 收集的其他檔案包括 Web 瀏覽器記錄和書籤、桌面和 USER 目錄中的 XLSX 和 CSV 檔案以及 MS-SQL 伺服器中的資料庫備份 (.bak) 檔案。

有趣的是，我們觀察到威脅行為者使用 WinRAR 來存檔相關文件，透過執行 certutil -encode 命令對存檔進行 Base64 編碼，然後運行 type 命令透過 Web shell 將 Base64 內容列印到螢幕上。

Unit 42 表示：“通過將檔案編碼為文本並將其打印到屏幕上，攻擊者無需實際上傳任何文件即可竊取數據。” “攻擊者可能選擇這種方法，因為主機上的 shell 允許他們運行命令和查看輸出，但不能直接傳輸文件。”

這些攻擊中採用的技術之一是使用合法的 Python 可執行檔（「python.exe」和「pythonw.exe」）來發起 DLL 側面載入攻擊並秘密執行惡意 DLL，包括用於持久存取的 FRP、PrintSpoofer 和名為 ScanPortPlus 的基於 Go 的自訂掃描程式。

據稱，早在 2020 年，CL-UNK-1068 就已經使用名為 SuperDump 的自訂 .NET 工具從事偵察工作。最近的入侵已轉變為使用批次腳本收集主機資訊並映射本地環境的新方法。

Unit 42 總結道：“該組織主要使用開源工具、社區共享的惡意軟體和批次腳本，在滲透關鍵組織的同時成功地維持了秘密行動。”

「這一系列活動透過跨 Windows 和 Linux 環境進行操作，為每個作業系統使用不同版本的工具集，展示了多功能性。雖然對關鍵基礎設施和政府部門的憑證盜竊和敏感資料外洩的關注強烈表明存在間諜動機，但我們還不能完全排除網路犯罪意圖。”
