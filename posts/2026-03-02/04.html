04. 🔗 2026-03-02 開源 AI：Anthropic 將 MCP 捐贈 Linux 基金會 Agentic AI Foundation OpenAI Google Microsoft AWS 聯合參與

全文翻譯（繁中）

MCP 正式捐贈 Linux 基金會：開放 AI 生態的重要里程碑

Anthropic 正式宣佈將其於 2024 年 11 月推出的 Model Context Protocol（MCP，模型上下文協議）捐贈予 Linux 基金會旗下新成立的 Agentic AI Foundation（AAIF，代理式 AI 基金會）。此舉標誌著 MCP 從 Anthropic 的私有開放標準，正式轉型為由業界共同治理的中立開放標準，Anthropic 承諾確保 MCP 保持中立性，不受任何單一公司的商業利益所左右。

Agentic AI Foundation：AI 巨頭史無前例的跨競爭協作

Agentic AI Foundation 由 Anthropic、Block（Square 母公司）及 OpenAI 三家公司共同創立，Google、Microsoft、AWS（亞馬遜雲服務）、Cloudflare 及 Bloomberg 亦作為支持成員參與其中。基金會的成立意義在於，它首次將原本互為競爭對手的頂尖 AI 公司凝聚於同一治理框架下，共同推動 AI 代理系統基礎設施標準的開放化發展，以透明、協作及符合公眾利益為核心原則。

MCP 的現狀：一年內成為業界事實標準

MCP 自 2024 年 11 月推出至今僅約一年，已取得令人矚目的採用成果。目前全球已有逾 10,000 個公開 MCP 伺服器在運行，涵蓋各類外部工具、資料庫及應用程式接入。在主流 AI 工具方面，ChatGPT、Cursor、Gemini 及 Visual Studio Code 均已原生支援 MCP，雲端基礎設施層面亦有 AWS、Cloudflare、Google Cloud 及 Microsoft Azure 提供企業級支援，Claude 本身亦透過 MCP 提供 75 個以上的外部系統連接器。

MCP 的核心價值：AI 的「USB-C」通用接口

MCP 的設計理念是為 AI 模型與外部工具、數據來源及應用程式之間的溝通提供統一的標準協議，被廣泛比喻為「AI 世界的 USB-C 接口」。在 MCP 出現之前，每個 AI 應用程式都需要為每個外部工具單獨開發整合方案，極大增加了開發成本和複雜度；MCP 的標準化令 AI 代理可以即插即用地連接任何兼容工具，大幅降低了構建複雜 AI 工作流程的門檻，使代理 AI 的能力邊界大幅擴展。

開源治理與未來展望

基金會承諾 MCP 項目的維護者將繼續以社群意見為優先，並保持決策過程的透明度，確保治理模式不因捐贈而改變。分析師認為，Agentic AI Foundation 的成立可能是 AI 行業走向互通互聯的關鍵轉捩點，若 MCP、AGENTS.md 及 Goose 等標準成為基礎設施，AI 代理生態或將從以封閉平台為主導，轉向如網際網路般開放互通的模組化生態系統，令開發者得以自由組合不同廠商的 AI 能力構建複雜應用。
