17. ⭐ 2026-02-23 GitHub AI 焦點：Perplexica 開源 AI 搜尋引擎累積 12K 星持續獲關注

全文翻譯（繁中）

Perplexica 作為知名 AI 搜尋引擎 Perplexity AI 的開源替代方案，其 GitHub 倉庫累積星數在近期突破 12,000 顆的里程碑，持續吸引全球技術社群的關注。Perplexica 的核心價值主張在於讓用戶能夠以自托管（self-hosted）方式部署一個功能完整的 AI 搜尋引擎，徹底掌控自己的搜尋數據和隱私，無需將搜尋查詢發送至任何第三方商業服務。項目採用 MIT 授權，允許任何人免費使用、修改和分發，包括用於商業目的。

在功能架構方面，Perplexica 實現了一套完整的「搜尋增強生成」（Search-Augmented Generation，SAG）流程。當用戶提交搜尋查詢時，系統首先對查詢進行意圖理解和關鍵詞優化，然後通過 SearXNG（一個開源的元搜尋引擎）從 Google、Bing、DuckDuckGo 等多個搜尋引擎同時獲取搜尋結果，再將相關網頁內容提取並整合為結構化的參考資料，最後調用 LLM 基於這些實時參考資料生成準確、有來源支撐的綜合回答。整個流程以 Next.js 前端和 Node.js 後端搭建，可透過 Docker Compose 一鍵部署。

Perplexica 最受用戶歡迎的特性之一是其靈活的 AI 後端支援。用戶可以選擇使用 OpenAI、Anthropic、Google 等雲端 API，也可以完全在本地環境中通過 Ollama 運行開源模型（如 Llama 4、Mistral 或 Phi-3），實現真正意義上的「端對端私隱保護搜尋」。在使用本地模型的情況下，所有搜尋查詢和 AI 推理均在用戶自己的設備上完成，不會有任何數據發送至外部服務器（除了觸發搜尋引擎查詢所必需的搜尋關鍵詞外）。這一特性對研究人員、記者、法律專業人士和任何需要處理敏感信息的用戶而言具有重要的實際意義。

在使用場景方面，自托管的 Perplexica 已被不少技術型企業和研究機構用於建立內部知識搜尋工具，將其與企業內部文件庫和知識庫整合，實現「AI 驅動的企業知識問答」功能。部分學術機構的研究小組亦利用 Perplexica 作為學術文獻研究助理，配合 arXiv、PubMed 等學術數據庫的 API 整合，快速從海量文獻中提取與研究課題相關的信息。Perplexica 的 API 接口設計亦允許開發者將其搜尋能力嵌入到更大型的 AI Agent 工作流程中。

在香港的採用情況方面，Perplexica 在本地技術社群中受到相當程度的關注，尤其是在《個人資料（私隱）條例》和數據合規要求較為嚴格的行業環境中。本地數位私隱倡議組織「數碼權利亞洲」近期將 Perplexica 列為其推薦的「私隱友善 AI 工具」名單之一，並提供了適合香港用戶的中文部署教程。值得注意的是，項目在繁體中文和粵語搜尋結果的質量上仍有改進空間，有貢獻者已在 GitHub 上提交了針對中文搜尋優化的 Pull Request，預計在下一個版本更新中合並。
