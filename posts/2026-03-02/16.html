16. ⭐ 2026-03-02 GitHub AI 焦點：MCP 正式成為 AI 工具通用標準 逾萬個公開伺服器運行 ChatGPT Cursor Gemini VS Code 全面採用

全文翻譯（繁中）

MCP：AI 世界的「USB-C 接口」正式確立

隨著 Anthropic 正式將 Model Context Protocol（MCP）捐贈予 Linux 基金會旗下新成立的 Agentic AI Foundation，MCP 作為連接 AI 模型與外部工具、資料庫及應用程式的通用開放標準地位得到全面確立。自 2024 年 11 月推出至今僅一年多，MCP 已成為 AI 開發生態中使用最廣泛的工具整合協議，被業界廣泛比喻為「AI 世界的 USB-C 接口」——讓任何 AI 應用無需定製開發即可即插即用地連接任何兼容工具。

逾萬個公開 MCP 伺服器：生態規模超越預期

據 Anthropic 官方公佈的數據，目前全球已有超過 10,000 個公開 MCP 伺服器在運行，涵蓋各類外部工具和服務的接入，包括網頁搜索、程式碼執行、資料庫查詢、雲端儲存、企業 CRM 系統及各類 API。這一生態規模遠超業界一年前的預期，反映開發者社群對 MCP 標準化價值的廣泛認可，以及對 AI 代理系統工具整合的龐大需求。

主流平台全面採用：ChatGPT 到 VS Code 的全覆蓋

在主要 AI 工具及開發環境方面，MCP 的採用已達到覆蓋主流平台的里程碑：OpenAI 的 ChatGPT、Anthropic 的 Claude（75 個以上內建連接器）、Google 的 Gemini 及廣受開發者歡迎的 AI 代碼編輯器 Cursor 均已原生整合 MCP 支援。微軟的 Visual Studio Code 亦將 MCP 支援整合至其 Copilot 功能中，令數以百萬計的開發者可直接在熟悉的開發環境中使用 MCP 接入外部工具。

企業基礎設施全面支撐：雲端巨頭齊落場

在雲端基礎設施層面，AWS、Google Cloud、Microsoft Azure 及 Cloudflare 均已宣佈提供企業級 MCP 基礎設施支援，允許企業在主要雲端平台上安全部署和管理 MCP 伺服器。這一「雲端巨頭背書」對企業採用 MCP 具有重要意義，意味著企業可以在受監管的雲端環境中構建符合安全及合規要求的 AI 代理工作流程，而無需自建和維護複雜的工具整合基礎設施。

對開發者的實際意義：代理 AI 應用開發門檻大幅降低

MCP 成為通用標準對全球（包括香港）開發者的最直接影響，是構建複雜 AI 代理應用的開發門檻大幅降低。過去，開發者為每個 AI 應用單獨整合外部工具需要耗費大量時間和精力；有了 MCP，開發者只需實現一次 MCP 接口，便可讓 AI 代理即插即用地使用任何兼容工具。GitHub 社群的反應印證了這一點——自 MCP 推出以來，基於 MCP 的開源項目數量呈指數級增長，香港本地開發者社群亦有不少 MCP 相關項目在 GitHub 活躍開發中。
