16. ⭐ 2026-01-13 GitHub AI 焦點：Mastra — TypeScript 優先的 AI Agent 框架

原文連結：https://github.com/mastra-ai/mastra/releases/tag/v2.0.0

全文翻譯（繁中）

Mastra 在 2026-01-13 的 GitHub 趨勢榜上持續攀升，主打 TypeScript 優先的 AI 代理開發框架。對於已熟悉 React/Next.js 生態的前端工程師而言，Mastra 提供了最低學習成本的 Agent 開發體驗。

框架支援工作流圖（Workflow Graph）定義複雜的多步驟任務、內建 RAG 管線支援知識庫檢索、工具呼叫機制讓代理可與外部 API 互動，以及多模型切換能力讓開發者根據任務複雜度自動選擇最適合的 LLM。

最新版本新增 MCP 伺服器支援與向量資料庫整合（Pinecone、Weaviate、Qdrant），讓開發者可在 5 分鐘內為代理添加記憶體與知識庫能力。社群已貢獻超過 200 個可重用的工具模組，覆蓋網頁搜尋、文件解析、郵件發送與資料庫操作。

在 GitHub 上單週新增超過 3,000 顆星，npm 每週下載量突破 50,000 次。多家 YC 孵化的初創已採用 Mastra 作為 Agent 後端框架，在客服自動化、數據分析與內容生成等場景中投入生產使用。
