19. 2026-04-13｜警告：甲骨文對人工智慧的痴迷可能意味著更高的價格和更糟糕的支持

全文翻譯（繁體中文・香港）

隨著拉里·埃里森（Larry Ellison）的公司做出巨額資料中心支出承諾以支持其人工智慧雄心，甲骨文客戶已被警告注意支援和定價方面的變化。

甲骨文談判顧問表示，支持水準正在下降，而隨著折扣消失，成本可能會增加。其他人則警告稱，合約談判將採取更加強硬的態度。

Oracle 正在積極擴展 AI 資料中心容量，以滿足與 OpenAI、xAI、Meta、Nvidia 和 AMD 等客戶和合作夥伴相關的需求。僅 OpenAI 的交易就預計在五年內價值約 3000 億美元。

隨著投資者越來越擔心甲骨文如何為其人工智慧資料中心的建設提供資金，同時大舉借貸以支持擴張，有關裁員影響數千名員工的廣泛報告出現。

今年 3 月，由於觀察家預計將進行更廣泛的裁員，甲骨文將其 2026 財年重組計畫的預計成本增加了 5 億美元，使總成本高達 21 億美元。

甲骨文先前籌集了 180 億美元的債務，此後概述了籌集更多資金的計劃，因為它為資料中心投資提供資金，旨在滿足人工智慧模型建構者和企業客戶不斷增長的需求。評級機構穆迪警告稱，隨著甲骨文加大人工智慧基礎設施投資，可能會出現進一步借貸和負自由現金流的情況。

商業顧問公司 House of Brick Technologies 的專業顧問技術長 Nick Walter 表示，有大量傳聞稱，Oracle 支援結果正在變得越來越糟，這意味著服務速度變慢、幫助技能不夠熟練，以及「更多的 Oracle 支援回應顯然是由人工智慧編寫的」。

他說，House of Brick 還發現最近有大量客戶尋求協助減少 Oracle 雲端基礎設施 (OCI) 使用量，或協助與 Oracle 就長期 OCI 交易進行談判。 “許多在過去三到五年內採用 OCI 的客戶突然發現，他們的續訂費用要高得多，因為 Oracle 取消或減少了企業折扣，他們被迫接近公開列出的即用即付價格。”

客戶還尋求協助以應對 Java 面臨的壓力，包括審計和銷售推動以及「聽起來很可怕的威脅」。

「這不是一個新現象，但壓力仍在繼續，Java 是 Oracle 與其客戶的錢包戰爭的主要戰線，」他說。

Palisade Discovery 的創辦人兼執行長 Craig Guarente 表示，Oracle 不再只是向客戶施加壓力，要求他們獲得 Java 許可；它正在進行正式審計。 「在過去的幾個月裡，我們確實看到了經典 Oracle 審計實踐的回歸，」他說。

2023 年，Oracle 更改了其 Java SE 訂閱模式，從按用戶或按處理器訂閱轉變為按員工訂閱。批評者稱此舉是“掠奪性的”，因為那些很少使用 Java 但擁有大量員工的組織可能會因成本增加而受到沉重打擊。同年晚些時候，Gartner 的研究表明，在新的授權模式下，使用相同軟體的成本可能會增加兩到五倍。

Guarente 也警告 OCI 合約續約時價格會上漲。 「如果你沒有得到充分的保護，那麼我們將獲得 9% 或 10% 的長期鎖定。這一切都可以追溯到最初的購買。如果你在談判這些合約時考慮周到，你就有一定的靈活性。但如果你簽署了當地標準協議，就沒有價格保護。”

Oracle 和 SAP 顧問公司 Remend 的創始合夥人 Eric Guyer 在社群媒體貼文中指出，軟體支援的利潤遠高於資料中心的利潤。

「這意味著您的軟體支援付款對甲骨文的價值正在不斷增加，而對您的價值卻在減少，而且速度在增加。我的意思是，這在技術上是透過創新的減弱和每年超過 8% 的通貨膨脹調整來實現的，」他說。

蓋耶告訴 The Register ，客戶也擔心失業對他們與甲骨文關係的長期影響。
