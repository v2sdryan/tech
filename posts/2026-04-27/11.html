11. 2026-04-26｜直接去賣吧！ Windows 二次安裝以 IT 部門為代價兜售 Microsoft 服務

全文翻譯（繁體中文・香港）

意見 您的筆記型電腦已經使用幾個月了，並且您始終確保它安裝了 Microsoft 更新。然後有一天，當你啟動時，Windows 11 會向你顯示一條令人困惑的消息：“你的電腦即將完成設定。”

快完成了嗎？這可能意味著什麼？您很久以前就在 Windows 11 上設定了計算機，但這裡它告訴您您的安裝從未真正完成。接下來你會發現還有哪些完成的任務是你其實還沒完成的呢？您是否可能忘記在五年前簽署結婚證書、在您居住了 10 年的房屋契約上簽名，或者錯過 2003 年參加的駕駛考試的關鍵時刻？您確定您孩子的出生證明上的名字是您的父母嗎？

您的 PC 螢幕設定即將完成 - 點擊放大

因此，您按一下對話方塊底部出現的「繼續」按鈕，Windows 會詢問您是否要使用 Microsoft 建議的瀏覽器設定。你不確定設定是什麼，但你點擊了「使用建議設定」按鈕，因為這一切聽起來都很光明正大，而且沒有「不使用推薦設定」按鈕（螢幕的另一部分有一個切換按鈕，但這很令人困惑）。目前尚不清楚這會產生什麼影響 - 在我們最近的測試中，它沒有將預設瀏覽器從 Google Chrome 更改為 Microsoft Edge，但您的情況可能會有所不同。

螢幕要求您使用建議的瀏覽器設定 - 點擊放大

接下來，您會看到一個螢幕，鼓勵您將手機與電腦連接起來，以便您可以在桌面上接收簡訊。這看起來很無辜，但您會看到並點擊“跳過”連結（而不是“下一步”按鈕，後者是“是”），因為您現在不想這樣做。

螢幕要求您設定手機與電腦的連線 - 點擊放大

然後，您會看到另一個螢幕，提醒您已安裝 Office，並讓您按一下另一個顯示「知道了」的按鈕。現在，您開始懷疑微軟是否與美國骨科醫師學會勾結，只是想讓您點擊，直到您患有腕管綜合症。但如果您沒有 MS Office，這可能是 Microsoft 試圖向您出售訂閱的地方。

如果您尚未安裝 Office 365，此畫面將嘗試向您出售 Office 365 - 按一下放大

Screen 試圖向您出售 Xbox Game Pass 訂閱 - 點擊放大

同樣，您可以隨意點擊“跳過”鏈接，儘管“加入只需 14.99 美元”按鈕在屏幕上更加突出。如果幸運的話，在此之後僅再出現一個螢幕，並且您有機會看到一些不必要的 Windows 提示，如果單擊“是”，它將在您的瀏覽器中啟動。

真正不必要的螢幕會詢問您是否需要一些 Windows 提示 - 點擊放大

您剛剛經歷的是 Windows 11 的第二次開箱即用體驗 (SCOOBE)，這是 Microsoft 的一種極具欺騙性和不道德的嘗試，旨在誘騙您購買更多月度服務或使用雷蒙德希望您擁有的設置。 SCOOBE 可以在您首次設定電腦後數月甚至數年啟動，並且您可以在電腦的生命週期中多次取得它，通常是在 Windows 更新後。這是字典中對enshittification的定義，赤裸裸地試圖誘騙你在你準備工作或娛樂時立即給微軟更多的錢——啟動——。

但 SCOOBE 不僅僅是一種煩惱和一種搶錢手段。對於擁有託管 PC 的組織來說，這是一個嚴重的問題。這種奇怪的設定模式花費的不僅是員工一分鐘的生產力。這會引發支援電話，因為不太懂科技的員工認為他們的工作電腦確實有問題。在與數十位 IT 人員的交流中，我聽到了許多關於因 SCOOBE 經歷而導致的支援請求和時間損失的故事。

如果您還沒有聽說過 SCOOBE，那麼當您長時間運行的電腦突然表示需要完成設定時，致電 IT 部門是有意義的。如果我收到它並且不了解情況，我會認為有人剛剛擦除了我的電腦並重新安裝了 Windows。我希望 IT 部門介入並取回我的所有數據。

「SCOOBE 在我們的設備配置幾個月後首次出現在我們的設備上。出現時間特別不方便，因為它們出現在我們貢獻者的螢幕上，這些貢獻者正在積極收集數據，並認為他們的機器可能存在技術問題，」人工智慧培訓數據公司 Unidata 的數據收集項目經理 Hanna Parkhots 告訴 The Register 。 “這導致大量的支援請求增加，我們透過查看一周內針對同一 SCOOBE 相關訊息提交的三張錯誤請求發現了這一點。”
