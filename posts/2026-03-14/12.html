12. 2026-03-13｜GitHub 從免費 Copilot 計畫中刪除一些模型，激怒了學生

全文翻譯（繁體中文・香港）

你不會得到你不付出的代價！微軟的 GitHub 正在削減開支，從其免費的 GitHub Copilot Student 計劃中刪除了幾個昂貴的高級模型。

週三，GitHub 開發者關係副總裁 Martin Woodward 爆料了這一不受歡迎的消息，導致與該網站學生開發者社群的關係惡化。

Woodward 將計劃變更描述為讓學生可持續訪問的努力，他表示，從 3 月 12 日星期四開始，Copilot 的免費訪問將根據新的 GitHub 學生計劃以及其他 GitHub Education 福利進行管理。

「然而，作為這一過渡的一部分，一些高級模型，包括 GPT-5.4、Claude Opus 和 Sonnet 模型，將不再可供 GitHub Copilot 學生計劃下的自我選擇，」他在討論論壇帖子中解釋道。 “我們知道這會令人失望，但我們正在做出這一改變，以便我們可以讓 Copilot 免費並可供世界各地數百萬學生使用。”

學生計畫仍可使用許多模型，包括 Claude 4.5 Haiku（正常價格 = 輸入 $1 / 1M 代幣；輸出 $5 / 1M 代幣用於輸出）、Gemini 3.1 Pro（輸入 $2 / 1M；輸出 $12 / 1M）和 GPT-5.3 Codex（輸入 $M.

OpenAI 和 Anthropic 沒有立即回應置評請求。

GitHub 的決定引起了憤怒。伍德沃德的帖子只獲得了 21 票贊成，而在本文提交時有 2,874 票反對，過去兩天收到了 1000 多條評論。

大多數評論都表示失望，並指出獲得表現特別好的模型可以帶來教育上的好處。

一位名為 Sahad Rushdi 的論壇參與者評論道：“對於我們許多從事高級工程項目的人來說，Claude 4.6 Sonnet 和 Opus 不僅僅是‘選擇’——它們是目前最有能力進行編碼、邏輯和處理大規模重構的人工智能代理。限制這些模型的自我選擇限制了我們學習行業領先技術的能力。”

這種觀點得到了 Nguyễn Thế Toàn 的個人發文的呼應：“刪除 GPT-5.4、Claude Opus 和 Claude Sonnet 等高級模型會使學習編程變得更加困難。這些模型更擅長解釋複雜的編碼概念，幫助調試問題，並在學生陷入困境時逐步指導學生。”

針對眾多要求恢復高階機型准入的呼聲，伍德沃德週四提出了一個建議：付費。

「我們現在添加了該選項，以便人們可以根據需要從 GitHub Copilot Student 計劃升級到付費 GitHub Copilot Pro 或 GitHub Copilot Pro+ 計劃，同時保留 GitHub Student Pack 的其餘權益，」他在最初帖子的更新中說道。
