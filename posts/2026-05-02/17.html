17. 2026-05-01｜OpenAI 將 GPT-5.5-Cyber​​ 鎖定在天鵝絨繩後面，儘管猛烈抨擊 Anthropic 這樣做

全文翻譯（繁體中文・香港）

OpenAI 正準備向精心挑選的「網路捍衛者」圈子限量發布其新的 GPT-5.5-Cyber​​ 模型，而就在幾週前，Anthropic 因做了幾乎完全相同的事情而遭到抨擊。

執行長 Sam Altman 在 X 上發表的一篇文章中表示，該推出將在「未來幾天內」開始，訪問權限僅限於他所描述的致力於保護關鍵系統的值得信賴的防御者群體。

他寫道：「我們將與整個生態系統和政府合作，找出可信賴的網路存取方式。」他補充說，目標是「迅速幫助保護公司和基礎設施的安全」。

GPT-5.5-Cyber​​ 旨在在其他人濫用缺陷之前發現缺陷。 OpenAI 表示，它可以進行滲透測試、發現錯誤、利用它們並分解惡意軟體，但正如我們已經看到的那樣，破壞系統的工具很少會長期掌握在正確的人手中。

OpenAI 宣布這一消息的幾週前，Anthropic 向大約 50 個受到嚴格控制的組織推出了自己的網路模型 Claude Mythos，並表示該模型永遠不會公開，而 Altman 對此並不以為然。

據 TechCrunch 報導，他在 Core Memory 播客上露面時，以謹慎的姿態瞄準了他所謂的排他性。

他說：「長期以來，世界上有人希望將人工智慧掌握在一小部分人的手中。」 「你可以用很多不同的方式來證明這一點。」他更進一步，將這種方法比作出售恐懼。 “我們製造了一枚炸彈，我們要把它扔到你頭上。我們將以 1 億美元的價格賣給你一個防空洞。”

現在，OpenAI 即使不建造同樣的避難所，至少也會在門口檢查身分證。

獨立測試表明該模型不僅僅是行銷廢話。英國人工智慧安全研究所本週表示，GPT-5.5-Cyber​​ 是“我們在網路任務中測試過的最強大的模型之一”，並指出它只是第二個完成端到端多步驟攻擊模擬的系統。

它可能被定位為保護，但當這些工具既可以破壞系統也可以修復系統時，差異通常取決於誰先到達那裡。 ®
