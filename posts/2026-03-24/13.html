13. Mon, 23 Ma｜有人公開洩漏了一個可以破解數百萬支 iPhone 的漏洞工具包

全文翻譯（繁體中文・香港）

上週，網路安全研究人員發現了一場針對 iPhone 用戶的駭客活動，該活動使用了一種名為 DarkSword 的高級駭客工具。現在有人洩漏了 DarkSword 的新版本，並將其發佈在程式碼共享網站 GitHub 上。

研究人員警告稱，這將使任何駭客能夠輕鬆使用這些工具來攻擊運行舊版蘋果作業系統且尚未更新到最新 iOS 26 軟體的 iPhone 用戶。根據蘋果自己關於過時設備的數據，這可能會影響數億活躍使用的 iPhone 和 iPad。

「這很糟糕。它們太容易改變用途了，」行動安全新創公司 iVerify 的聯合創始人 Matthias Frielingsdorf 週一告訴 TechCrunch。 “我認為這種情況已經無法再得到遏制。因此，我們需要預計犯罪分子和其他人會開始部署這一點。”

Frielingsdorf 表示，這些新版本的 DarkSword 間諜軟體與他和他的 iVerify 同事先前分析的版本共享相同的基礎設施，儘管檔案略有不同。他說，上傳到 GitHub 的檔案並不複雜，只有 HTML 和 JavaScript，這意味著任何人都可以「在幾分鐘到幾小時內」複製並貼上它們並將它們託管在伺服器上。

「這些漏洞將立即發揮作用，」弗里林斯多夫說。 “不需要 iOS 專業知識。”

先前分析過 DarkSword 漏洞的Google發言人 Kimberly Samra 表示，該公司的研究人員同意 Frielingsdorf 的評估。

聯絡我們 您是否有關於 Darksword、Coruna 或其他政府駭客和間諜軟體工具的更多資訊？在非工作裝置上，您可以透過 Signal（電話 +1 917 257 1382）或透過 Telegram、Keybase 和 Wire @lorenzofb 或電子郵件安全地聯絡 Lorenzo Franceschi-Bicchierai。

一位名為 matteyeux 的安全愛好者也告訴 TechCrunch，使用洩漏的 DarkSword 樣本確實是微不足道的。 Matteyeux 在 X Monday 的一篇貼文中寫道，他能夠使用網路上流傳的「野外」DarkSword 範本破解運行 iOS 18 的 iPad mini 平板電腦，iOS 18 是容易受到 DarkSword 攻擊的上一代作業系統。

蘋果發言人 Sarah O’Rourke 告訴 TechCrunch，該公司已意識到針對運行舊版和過時操作系統的設備的漏洞，並於 3 月 11 日針對無法運行最新版本 iOS 的設備發布了緊急更新。

奧羅克表示：「為了維護 Apple 產品的安全性，保持軟體更新是最重要的一件事。」他補充說，安裝了更新軟體的設備不會受到這些報告的攻擊的威脅，而且鎖定模式也會阻止這些特定攻擊。

擁有 GitHub 的微軟發言人沒有立即回應置評請求。

TechCrunch 未連結到該程式碼，因為它可用於主動攻擊，其中包含一些註釋，描述了漏洞利用的工作原理以及如何實施它們。

一則評論（可能是由參與 DarkSword 的開發人員之一撰寫）表示，該漏洞“通過 HTTP 從 iOS 設備讀取並竊取與取證相關的文件”，指的是從某人的 iPhone 或 iPad 竊取信息，然後通過互聯網將數據發送到攻擊者控制的伺服器。

「這個有效負載應該被注入到具有檔案系統存取類別的進程中，」評論中寫道。

在一種情況下，程式碼引用了“後利用活動”，並描述了惡意軟體獲取用戶手機存取權限並獲取其中內容（包括聯絡人、訊息、通話記錄和 iOS 鑰匙圈）後的過程，其中 iOS 鑰匙串存儲 Wi-Fi 密碼和其他機密，並將其轉儲到遠端伺服器中。

另一個文件包含向烏克蘭一家流行服裝網站上傳資料的參考，但 TechCrunch 無法立即確定原因。據稱，俄羅斯政府駭客使用「暗劍」攻擊烏克蘭目標。
