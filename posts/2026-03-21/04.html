04. Fri, 20 Ma｜微軟在 Windows 上消除了 Copilot AI 的部分臃腫

全文翻譯（繁體中文・香港）

微軟週五宣布了一系列旨在提高 Windows 11 作業系統品質的變化，其中特別包括減少人工智慧助理 Copilot 的入口點數量。

該公司表示，將減少某些應用程式中 Copilot AI 的集成，首先是照片、小部件、記事本及其截圖工具。

Windows 和裝置部門執行副總裁 Pavan Davuluri 在「將人工智慧整合到最有意義的地方」的標題下，在公司部落格上寫道，微軟正在更加關注「Copilot 在 Windows 上的整合方式和位置」。他解釋說，其目標是專注於「真正有用」的人工智慧體驗。

這種將人工智慧整合到現有平台的「少即是多」的方法可能反映了消費者對人工智慧膨脹的日益強烈的抵制。雖然當今許多人都認為人工智慧是一種有用的工具，但也存在著對信任和安全的擔憂。例如，皮尤研究中心本月發布的一項研究指出，截至 2025 年 6 月，一半的美國成年人現在對人工智慧的擔憂多於興奮，高於 2021 年的 37%。

這並不是微軟第一次重新考慮其 Copilot 整合。本月早些時候，新聞網站 Windows Central 表示，該公司在 Windows 11 上提供 Copilot 品牌的人工智慧功能的計劃已被悄悄擱置。該網站表示，這包括「設定」應用程式、檔案總管和其他地方的一些系統級整合。

在此之前，微軟已將其人工智慧驅動的記憶體功能「Windows Recall for Copilot + PC」的推出推遲了一年多，因為它試圖解決用戶的隱私問題。召回功能於去年四月推出，但安全漏洞仍在被發現。

顯然，用戶回饋正在影響微軟在 Windows 上圍繞人工智慧採取的行動。 Davuluri 寫道，他和他的團隊在過去幾個月裡一直在傾聽社群的意見，了解他們希望看到 Windows 如何改進。

Copilot 回滾只是正在進行的變更之一。

該公司表示，它還推出了將工作列移動到螢幕頂部或兩側的功能，讓用戶可以更好地控制系統更新，加快文件資源管理器的速度，改進小部件體驗，更新反饋中心，並使其更容易瀏覽其 Windows Insider 計劃——一個提供有關 Windows 未來反饋的社區。
