13. Mon, 27 Ap｜OpenAI 結束了微軟因其 500 億美元亞馬遜交易而面臨的法律風險

全文翻譯（繁體中文・香港）

2026 年首屆 StrictlyVC 將於 4 月 30 日登陸舊金山。門票火熱搶購。立即註冊。

週一，微軟和 OpenAI 宣布，他們再次就約束兩家公司的交易進行了重新談判。儘管一些關於 X 的觀點將其視為 ChatGPT 製造商對 Windows 巨頭的勝利，但雙方都將成為贏家。

最重要的是，新條款解決了 OpenAI 與亞馬遜簽署高達 500 億美元協議以來一直困擾其的問題。

透過這項新協議，微軟不再擁有 OpenAI 所有產品和 IP 的獨家存取權，直到 OpenAI 產生 AGI 的神奇日子到來，而是其合作夥伴關係有了明確的時間表。該合約為微軟提供了截至 2032 年模型和產品 OpenAI IP 的非獨佔許可。

兩家公司仍然稱微軟 OpenAI 為“主要雲端合作夥伴”，這意味著 OpenAI 的大部分雲端服務可能會在本次交易涵蓋的六年內由 Azure 提供服務，儘管 OpenAI 急於與其他合作夥伴建立自己的資料中心。 10月，OpenAI同意額外購買價值2,500億美元的微軟雲端服務。這句話是向微軟股東傳達的訊息：OpenAI 仍將是 Azure 的龐大客戶。

兩家公司表示，OpenAI 產品將「首先在 Azure 上發布，除非微軟不能並且選擇不支援必要的功能」。但至關重要的是，“OpenAI 現在可以通過任何雲端提供商向客戶提供其所有產品。”

再次強調，本公告中並沒有明確定義“第一”，這是否意味著僅在一段時間內在 Azure 上獨佔，或者僅僅意味著微軟也將成為搭載 OpenAI 最新產品的供應商之一。

但這個術語最重要的部分是：它解決了微軟可能因人工智慧實驗室與亞馬遜的交易而起訴 OpenAI 的可能性。

回顧一下這一混亂局面：2月份，OpenAI宣布亞馬遜將向這家模型製造商投資高達500億美元，其中包括150億美元的初始投資和「在未來幾個月滿足某些條件時」另外投資350億美元，公司表示，但沒有具體說明這些條件是什麼。

作為交換，OpenAI 同意在 AWS Bedrock（提供各種 AI 模型和服務的 AWS 服務）上共同開發「有狀態運行時技術」。有狀態運行時是支援人工智慧代理的技術，使它們能夠長時間記住任務和上下文。

OpenAI 也承諾 AWS 將擁有提供 OpenAI 的新代理商製作工具 Frontier 的獨家權利。這就是問題所在。

OpenAI 與 Microsoft 的最初協議阻止 OpenAI 在 AWS 上獨家銷售 Frontier，並且可能會阻止 AWS 完全銷售它。

雖然微軟先前同意讓 OpenAI 在其他雲端供應商上運行某些特定產品，例如消費者 ChatGPT，但它保留了透過 API 存取的任何 OpenAI 產品（例如 Frontier）的專屬權。

事實上，就在 OpenAI 宣布與 AWS 達成交易的同一天，微軟公開反駁了 AWS 的獨家條款，並寫道（重點是微軟的）：

Microsoft 保留其 OpenAI 模型和產品的獨家授權和智慧財產權存取權。 ...Azure 仍然是無狀態 OpenAI API 的獨家雲端供應商。 ... OpenAI 與任何第三方（包括 Amazon）合作產生的對 OpenAI 模型的任何無狀態 API 呼叫都將託管在 Azure 上。 ... OpenAI 的第一方產品，包括 Frontier，將繼續託管在 Azure 上。

微軟也強調，其條款在 OpenAI 實現 AGI 之前一直有效。英國《金融時報》報道稱，如果微軟必須執行這些合約條款，它甚至考慮採取法律行動。
