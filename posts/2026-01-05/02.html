2. 🔮 2026-01-05 AI 模型：Anthropic Claude Opus 4.5 百萬 token 上下文窗口

原文連結：https://www.anthropic.com/research/claude-opus-4-5-2026-01-05

全文翻譯（繁中）

Anthropic 正式推出 Claude Opus 4.5，標誌性地支援百萬 token 上下文窗口（約 75 萬個中文字），為處理超長文檔與複雜多步推理設定了新的行業標準。這一突破讓 AI 可以在單次對話中完整處理一本小說、一份完整的法律合約集或一個中型程式碼庫。

百萬 token 上下文窗口的技術實現依賴於 Anthropic 獨創的「階層注意力機制」（Hierarchical Attention）。傳統 Transformer 的注意力機制隨序列長度呈二次方增長，而 Anthropic 的新架構將長文檔分層處理——先提取段落級摘要，再在需要時回溯細節，記憶體使用量僅為傳統方法的 1/16。

在實際應用測試中，Claude Opus 4.5 展現了驚人的長文理解能力。在「大海撈針」測試中（在百萬 token 文本中找出特定資訊），準確率達到 99.2%。在法律合約分析中，模型能同時比較 50 份不同版本的合約，自動標註差異條款並評估風險等級。

多代理協作是 Claude Opus 4.5 的另一大亮點。Anthropic 推出了全新的「Cowork」功能，讓多個 Claude 實例可以各司其職——一個負責閱讀文檔、一個負責撰寫摘要、一個負責事實核查。在企業客戶的早期測試中，這種分工模式讓複雜報告的生成效率提升了 5 倍。

定價方面，Claude Opus 4.5 的 API 每百萬輸入 token 定價為 $15，輸出為 $75。雖然價格較高，但考慮到百萬 token 上下文窗口帶來的能力提升，實際使用中可減少多次 API 呼叫的需要，總體成本可能更低。Anthropic 同時推出了 Claude Pro 訂閱方案，月費 $20 可獲得較高的使用量。

Replit、Asana 與多家律師行的早期測試者報告，Claude Opus 4.5 在處理複雜、需要深度理解的任務時表現尤為出色。一位國際律師行合夥人表示：「我們過去需要初級律師花三天時間閱讀的合約文件，現在 Claude 可以在 10 分鐘內完成分析，而且準確度更高。」
